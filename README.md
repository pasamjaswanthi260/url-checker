# URL Safety Checker

This is a simple URL Safety Checker website.

## Description
The website checks whether a given URL looks safe or unsafe using basic rules.

## Features
- Checks HTTPS
- Checks URL length
- Finds suspicious symbols
- Detects IP address URLs

## Technology
- HTML
- CSS
- JavaScript

## How to Run (Localhost)
- This project runs on localhost (local system)
- No backend and no database are used

### Steps
1. Download the project
2. Open `index.html` in a browser
3. Enter a URL and click **Check URL**

## Note
This project is for learning purpose only.
